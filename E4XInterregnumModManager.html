Enhanced 4X Mod/Star Wars Interregnum

Mod Configuration Manager

This page will generate the EnabledMods.txt file for the Enhanced 4X Mod and/or Star Wars: Interregnum. Simply select which mod and minimods you want to use, and it will take care of the rest.

Step 1: First select your main mod.

Step 2: Select which Gameplay Minimods you wish to use, if any.

Step 3: Select which Graphics Mods you wish to use, if any. Note: Sins: Remastered requires a separate download if you wish to use it.

Step 4: Copy the output below to your EnabledMods.txt file. This should exist in the same directory that you extract mods in.
